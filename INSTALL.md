# Installation

## Prerequisites

- Git
- Node.js 18+ (recommended)
- Access to Claude Code or compatible AI agent

## Setup

1. Clone the repository:

   ```bash
   git clone <repo-url> AI-Engineering-OS
   cd AI-Engineering-OS
   ```

2. (Optional) Initialize the memory index:

   ```bash
   touch brain/MEMORY.md
   ```

3. Start using the OS with your AI agent.
