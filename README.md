# AI Engineering OS

An operating system for AI software engineering.

Instead of behaving like a chatbot,
the AI behaves like an engineering organization.

It understands architecture.

Plans before coding.

Maintains project memory.

Reviews its own work.

Writes tests.

Tracks decisions.

Updates documentation.

Learns project architecture.

Optimizes continuously.

Can be installed into any repository.

## Structure

| Path | Purpose |
|------|---------|
| `brain/` | Memory, knowledge base, persistent context |
| `agents/` | Agent definitions and configurations |
| `skills/` | Reusable skill modules |
| `rules/` | Rules and constraints for agent behavior |
| `templates/` | Template files for bootstrapping |
| `workflows/` | Multi-step workflow scripts |
| `docs/` | Documentation and guides |
